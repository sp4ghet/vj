## Requirements

- Unity 2018.2
- Windows
- OBS Studio with obs-ndi plugin
- Atom + [Veda](https://github.com/fand/veda)


## Install

```shell
git clone https://github.com/sp4ghet/vj
git submodule update --init
```

Open in Unity.
